React Native is...

...is an open-source JavaScript framework created by Meta (formerly Facebook) that enables developers to build truly native mobile apps for iOS and Android using a single codebase.

React Native applies React’s component-based model to native views, enabling a single codebase to produce platform-specific interfaces

History

  • “Biggest mistake… betting too much on HTML5” – Mark Zuckerberg
  • Launched in 2015 as Facebook moved away from unstable HTML5

True React Native Limitations

Performance Overhead

The legacy asynchronous bridge can introduce frame drops under CPU-heavy tasks.

Incomplete “Write Once, Run Anywhere”

Many native features still require custom native modules or third-party plugins.

Animation & Multithreading

Out of the box, JavaScript runs on a single thread—complex animations may slow down.

Dependency Management

The ecosystem’s wealth of packages—including some from Software Mansion—brings maintenance challenges when libraries break on major React Native upgrades.
